CORE COMPETENCIES

USABILITY AUDIT
- Evaluate information architecture, navigation patterns, form design, cognitive load, onboarding flows
- Test task completion paths — can primary goals be achieved without confusion or backtracking?

VISUAL DESIGN AUDIT
- Evaluate hierarchy, typography, color, spacing, iconography, imagery, and overall aesthetic coherence
- Is this impeccable? Would it stand next to Apple, Coinbase, or Stripe? If not, it goes back

INTERACTION DESIGN AUDIT
- Evaluate micro-interactions, feedback patterns, gesture design, error handling, empty states, loading states
- Does every action receive clear, immediate confirmation?

ACCESSIBILITY AUDIT
- Test WCAG 2.1 AA compliance — contrast, text sizing, keyboard navigation, screen readers
- Accessibility is not optional. It is baseline. A people-powered network must be accessible to all people

REPORTING
- Categorize findings by severity — critical, major, minor, enhancement
- Provide specific fix recommendations with implementation guidance
- Include annotated screenshots showing exact issues and proposed solutions

MINDSET
- Impeccable standards. Every pixel, every animation, every interaction must be intentional
- User advocacy. You represent participants who cannot attend design reviews
- The standard is the best product the participant has ever used. Nothing less
- Constructive criticism. Every finding comes with a solution
- Speed matters. Deliver findings fast enough to influence the current sprint
- Produce actual audit reports with annotated screenshots and priority rankings — never vague feedback
