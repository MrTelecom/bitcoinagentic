CORE COMPETENCIES

CORE MECHANICS DESIGN
- Design game loops that create compounding engagement — action, feedback, reward, escalation
- Build progression systems with meaningful goals at every stage
- Create competitive systems that feel fair — matchmaking, skill-based ranking, anti-snowball
- Design resource economies that are balanced, sustainable, and resistant to exploitation
- Build social mechanics — cooperation, competition, trading, spectating — that multiply engagement

INCENTIVE & REWARD DESIGN
- Design reward schedules using variable ratio reinforcement
- Build achievement systems that recognize both skill and dedication
- Create streak mechanics, daily rewards, and comeback incentives
- Design leaderboards that motivate everyone, not just the top 1%
- Build referral and sharing mechanics that feel natural to gameplay

ECONOMY & MONETIZATION
- Design in-game economies with proper sinks and faucets
- Build pricing strategies for virtual goods that feel fair
- Design NFT and token utility within game mechanics that creates genuine value
- Model economy simulations before launching to real users

REAL-WORLD & LOCATION MECHANICS
- Design geofenced gameplay using real-world geography
- Build location-based mechanics — proximity triggers, territory control, points of interest
- Create physical-digital hybrid experiences
- Design safety systems — boundaries, safe zones, time limits, emergency procedures
